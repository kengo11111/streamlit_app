"授業で扱っていない部品一覧"

1. startswith('M') は、文字列が指定した接頭辞で始まっているかどうかを判定するために使用した。本アプリでは、平均時間を表す列名がすべて「M」から始まっているため、このメソッドを用いることで必要な列のみを抽出している。これにより、分析対象外の列を除外し、後続のデータ処理を簡潔にしている。本アプリでは、CSV ファイルを読み込んだ直後に、平均時間の列を抽出する処理で使用している。

2. melt は、横持ち形式のデータを縦持ち形式に変換するために使用した。元のデータでは、年齢層や性別ごとの平均時間が複数の列に分かれて格納されていたため、そのままでは可視化や条件抽出が行いにくい。melt を用いることで、平均時間を一つの列として整理し、柔軟な分析が可能となった。本アプリでは、データを可視化用の形式に変換する段階で使用している。

3. Series は、正規表現によって抽出した項目名、年齢層、性別の情報を、それぞれ独立した列として追加するために使用した。関数の戻り値であるタプルを Series に変換することで、1行分のデータとしてデータフレームに展開することができる。本アプリでは、列名に含まれる属性情報を整理し、新しい列として追加する処理で使用している。

4. astype(int) は、データ型を整数型に変換するために使用した。調査年は文字列として読み込まれているため、そのままでは数値として扱うことができない。そこで文字列から「年度」を除去したうえで整数型に変換し、年次推移の可視化や回帰分析が正しく行えるようにした。本アプリでは、散布図の横軸に使用する調査年を処理する部分で使用している。
